# PostgreSQL-cheatsheet
A PostgreSQL cheat sheet with the common PostgreSQL commands

## General
### Switch and connect
<em>Run in terminal</em>
```
$ sudo -u postgres psql
```

### List all databases
<em>Run in psql console</em>
```
postgres=# \l
```

### Connect to the database named postgres
<em>Run in psql console</em>
```
postgres=# \c postgres
```

### Disconnect
<em>Run in psql console</em>
```
postgres=# \q
postgres=# \!
```
